# Bowmark

A mobile archery game built with gyro controls.

## Core Loop

- **Draw**: Tap and hold to draw the bow
- **Aim**: Tilt phone to aim (gyro/motion controls)
- **Release**: Lift finger to fire

## Progression

- Earn currency from hits & trick shots
- Upgrade stamina (longer aim time)
- Unlock new bows

## Target Types

- Static (training)
- Moving (timing)
- Trick shots (obstacles, distance)

## Environmental Factors

- Wind direction & strength
- Stamina drain while aiming

---

Built with React Native + Expo.
